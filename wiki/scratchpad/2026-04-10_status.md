## What Just Happened

The NBER Microdata 1989-1994 re-run with the `ormoth` fix completed successfully. This was the last outstanding download.

### The Bug That Was Fixed

In NBER microdata files for 1989+, the `origm` field changed from detailed Hispanic origin coding (0=Non-Hispanic, 1-5=Hispanic subcategories) to a binary flag. The script was using `origm` for all years, causing all White Non-Hispanic (WNH) counts for 1989-1994 to be 0. The fix: use `ormoth` for 1989+ files (same coding as pre-1989 `origm`).
